\section{Dataset}

Dataset pertama adalah Pima Indians Diabetes yang diperoleh dari Kaggle (\url{https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database}). Dataset ini berisi rekam medis pasien perempuan Pima Indian untuk diagnosis diabetes. Dataset memiliki 768 instansi, 8 fitur numerik, dengan 500 data mayoritas (non-diabetes) dan 268 data minoritas (diabetes), rasio ketidakseimbangan 1:1,87.

Dataset kedua adalah Haberman's Survival yang diperoleh dari UCI \textit{Machine Learning Repository} (\url{https://archive.ics.uci.edu/dataset/43/haberman+s+survival}). Dataset ini berisi data kelangsungan hidup pasien operasi kanker payudara. Dataset memiliki 306 instansi, 3 fitur numerik, dengan 225 data mayoritas (bertahan hidup) dan 81 data minoritas (tidak bertahan hidup), rasio ketidakseimbangan 1:2,78.

Dataset ketiga adalah Blood Transfusion Service Center yang diperoleh dari UCI \textit{Machine Learning Repository} (\url{https://archive.ics.uci.edu/dataset/176/blood+transfusion+service+center}). Dataset ini berisi data donor darah dari Pusat Layanan Transfusi Darah di Hsin-Chu, Taiwan. Dataset memiliki 748 instansi, 4 fitur numerik, dengan 570 data mayoritas (tidak mendonasikan) dan 178 data minoritas (mendonasikan), rasio ketidakseimbangan 1:3,20.

Dataset keempat adalah Glass5 dari \textit{UCI Glass Database} (\url{https://archive.ics.uci.edu/dataset/144/glass+identification}). Dataset ini berisi analisis komposisi kimia dari berbagai jenis kaca dari bangunan-bangunan di Inggris, digunakan untuk klasifikasi jenis kaca berdasarkan kandungan oksida logam. Dataset memiliki 214 instansi, 9 fitur numerik (tipe soda lime float glass, soda lime non-float glass, lead crown glass, lead silicate glass, dan window/wall glass), rasio ketidakseimbangan 1:22,78.
